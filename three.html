2. Introduction to CSS

In this topic, I learned about CSS (Cascading Style Sheets) and its role in controlling the presentation and layout of web pages. I gained knowledge about CSS selectors, properties, values, and how to apply them to HTML elements to style the webpage.

I also learned how to use CSS to add styles to HTML elements. I learned about CSS selectors, classes, and IDs, and how to apply various styles such as colors, backgrounds, fonts, margins, and padding to create visually appealing web pages.
